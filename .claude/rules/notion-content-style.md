# Notion 콘텐츠 작성 스타일

## 작성 절차

1. **내용을 먼저 정리** — 사용자에게 텍스트로 보여주고 검토
2. **Notion 마크다운 스펙 검토** — `notion://docs/enhanced-markdown-spec` 리소스를 조회하여 문법 확인
3. **검토 완료 후 적용** — 사용자 승인 후 Notion API 호출

## 페이지 구조 템플릿

```
## 개요
한두 문장으로 작업 목적 요약.

## 변경 내용
### 세부 항목 1
- 변경 사항 목록

### 세부 항목 2
- 변경 사항 목록

## 관련 파일
- `파일 경로`
```

## 주의사항

- 콘텐츠 문자열에 `\n` 이스케이프를 사용하지 말 것 — **실제 줄바꿈** 사용
- 테이블은 파이프(`|`) 문법이 아닌 **`<table>` 태그** 사용 (Notion 스펙)
- 콘텐츠는 **한국어**로 작성
- 페이지 제목은 content에 포함하지 않음 (properties의 title로 관리)
