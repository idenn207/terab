## v0.1.0 — Core Platform (Must 28개)

### 목표

NAS Drive의 핵심 기능을 갖춘 첫 번째 릴리스. 인증(Push 2FA), 모바일 앱, 알림 마이크로서비스, 기본 파일 관리, 공유 링크, 관리자 기본 기능을 포함한다.

### 범위 조정 사항

- **ADMIN-18**: 기본 역할(OWNER/ADMIN/USER) 목록 조회 + 역할별 권한 확인만 포함. 커스텀 역할 편집(ADMIN-17)은 v0.1.2에서 활성화
- **FILE-06**: 소프트 삭제 + D-09 기본 휴지통(목록/복원/영구삭제) 포함. TRASH-01 UI 고도화(필터, 정렬, 일괄 처리)는 v0.1.1
- **D-10**: v0.1.0에서는 프로필 조회만. 닉네임/비밀번호 변경(AUTH-04)은 v0.1.1
- **A-06**: v0.1.0에서는 기본 역할 목록 읽기 전용. 커스텀 역할 생성/수정(A-07)은 v0.1.2
