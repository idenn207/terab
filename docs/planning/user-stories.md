# NAS Drive User Story Map

## Activity 1: 파일 관리

```
[파일 관리] ──────────────────────────────────────────────
  ├─ 업로드        → "파일을 드래그해서 올린다"
  │                → "여러 파일을 한번에 올린다"
  │                → "업로드 진행률을 본다"
  ├─ 다운로드      → "파일을 클릭해서 받는다"
  │                → "여러 파일을 ZIP으로 받는다"
  ├─ 폴더 정리     → "새 폴더를 만든다"
  │                → "파일을 다른 폴더로 옮긴다"
  │                → "이름을 변경한다"
  ├─ 검색/탐색     → "파일명으로 검색한다"
  │                → "즐겨찾기한 파일을 본다"
  │                → "최근 파일을 본다"
  └─ 삭제/복구     → "파일을 삭제한다 (휴지통)"
                   → "휴지통에서 복원한다"
```

## Activity 2: 공유

```
[공유] ──────────────────────────────────────────────────
  ├─ 링크 공유     → "공유 링크를 생성한다"
  │                → "만료일/비밀번호를 설정한다"
  │                → "링크를 복사해서 전달한다"
  ├─ 사용자 공유   → "특정 사용자에게 권한을 부여한다"
  │                → "나에게 공유된 항목을 본다"
  └─ 외부 접근     → "링크로 접속해서 파일을 받는다 (비로그인)"
```

## Activity 3: 계정

```
[계정] ─────────────────────────────────────────────────
  ├─ 인증          → "이메일+비밀번호로 로그인한다"
  │                → "초대 링크로 가입한다"
  └─ 프로필        → "닉네임/비밀번호를 변경한다"
```

## Activity 4: 관리 (admin.drive.skypark207.com)

```
[관리] ─────────────────────────────────────────────────
  ├─ 사용자 관리   → "사용자 목록을 조회한다"
  │                → "새 사용자를 초대한다"
  │                → "사용자를 비활성화/삭제한다"
  │                → "관리자 역할을 부여한다"
  │                → "사용자별 사용량을 확인한다"
  ├─ 스토리지      → "전체 스토리지 현황을 본다"
  │                → "사용자별 용량 제한을 설정한다"
  ├─ 공유 관리     → "전체 공유 링크를 조회한다"
  │                → "문제 링크를 강제 만료한다"
  ├─ 시스템        → "CPU/메모리/디스크를 모니터링한다"
  │                → "서비스 상태를 확인한다"
  ├─ 설정          → "회원가입 정책을 변경한다"
  │                → "서비스 기본값을 설정한다"
  └─ 감사          → "사용자 활동 로그를 조회한다"
                   → "활동 통계를 본다"
```
