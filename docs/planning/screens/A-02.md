# A-02 대시보드

## 화면 개요

- 목적: 서비스 전반의 핵심 지표를 한눈에 파악하고 빠른 관리 액션 수행
- 진입 경로: `/dashboard` (로그인 후 기본 화면, 사이드바 "대시보드" 클릭)
- 사용자: ADMIN 역할 보유 사용자 (페르소나: 민수)
- 레이아웃: **Layout-B** (Admin Web 기본)
- 플랫폼: PC Web 전용
- 관련 요구사항: ADMIN-06
- 관련 유저플로우: Flow 6 "관리자 — 사용자 및 권한 관리" — 대시보드 진입

## 구성 요소

### Page Header

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 페이지 제목 | Text (h1) | "📊 대시보드" | - |

### 요약 카드 그리드

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 전체 사용자 카드 | StatCard | 총 사용자 수, 활성/비활성 내역 | 클릭 → A-03 이동 |
| 활성 사용자 카드 | StatCard | 최근 24시간 접속 사용자 수, 전체 대비 비율 | 클릭 → A-03 (활성 필터 적용) |
| 스토리지 사용률 카드 | StatCard | 프로그레스 바 + 사용량/총용량, 백분율 | 클릭 → A-08 이동 |
| 활성 공유 링크 카드 | StatCard | 활성 공유 링크 수, 만료 예정(7일 이내) 수 | 클릭 → A-09 이동 (그룹 5) |

### 빠른 액션

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 사용자 초대 버튼 | Button (primary) | A-05(사용자 초대)로 이동 | 기본 / 호버 |
| 스토리지 상세 버튼 | Button (secondary) | A-08(스토리지 대시보드)로 이동 | 기본 / 호버 |

### 최근 활동 테이블

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 테이블 헤더 | TableHeader | 시각 · 사용자 · 활동 유형 · 대상 | - |
| 활동 행 | TableRow | 최근 활동 10건 표시 | 기본 |
| 활동 유형 | Badge | 파일 업로드/다운로드/삭제, 폴더 생성, 공유 링크 생성/삭제, 로그인, 사용자 초대 등 | 유형별 색상 |
| 전체 로그 링크 | Link | "전체 활동 로그 보기 →" → A-14(감사 로그, 그룹 5) | 기본 |

## 인터랙션 상세

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 페이지 진입 | 4개 요약 카드 + 최근 활동 데이터 로드 |
| 2 | 요약 카드 클릭 | 해당 상세 화면으로 이동 (A-03, A-08, A-09) |
| 3 | 사용자 초대 클릭 | A-05(사용자 초대)로 이동 |
| 4 | 스토리지 상세 클릭 | A-08(스토리지 대시보드)로 이동 |
| 5 | 전체 활동 로그 보기 클릭 | A-14(감사 로그)로 이동 |
| 6 | 활동 행의 사용자 이름 클릭 | A-04(사용자 상세)로 이동 |

## 상태 목록

| # | 상태 | 설명 |
|---|------|------|
| 1 | 기본 | 요약 카드 + 최근 활동 정상 표시 |
| 2 | 로딩 | 요약 카드 스켈레톤 + 테이블 스켈레톤 |
| 3 | 부분 에러 | 일부 카드 데이터 로드 실패 — 해당 카드에 "데이터를 불러올 수 없습니다" + [다시 시도] |
| 4 | 전체 에러 | 페이지 전체 로드 실패 — 중앙 에러 메시지 + [다시 시도] |

## 반응형 동작

| 브레이크포인트 | 변경 사항 |
|----------------|-----------|
| Desktop (1024px+) | Sidebar 240px + 요약 카드 2×2 그리드 |
| Tablet (768~1023px) | Sidebar 접힘, 요약 카드 2×2 유지 |
| Mobile (< 768px) | Sidebar 숨김, 요약 카드 1열 스택, 테이블 가로 스크롤 |

## 접근성

- 요약 카드: `role="link"`, `aria-label` (카드명 + 수치 포함)
- 활동 테이블: `role="table"`, 각 행에 aria-label
- 프로그레스 바 (스토리지): `role="progressbar"`, `aria-valuenow`, `aria-valuemax`
- 빠른 액션 버튼: 명확한 aria-label
