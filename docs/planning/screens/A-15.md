# A-15 활동 통계

## 화면 개요

- 목적: 서비스 활동 지표(업로드 수, 활성 사용자, 공유 링크 생성)를 일별/주별 차트로 시각화
- 진입 경로: `/audit/stats` (사이드바 "감사 로그" → 서브메뉴 "통계")
- 사용자: `audit:read` 권한 보유 관리자 (페르소나: 민수)
- 레이아웃: **Layout-B** (Admin Web 기본)
- 플랫폼: PC Web 전용
- 관련 요구사항: ADMIN-16
- 우선순위: **Could** (v0.2 이후 구현 가능)

## 인터랙션 상세

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 페이지 진입 | 최근 7일 통계 데이터 로드 (요약 카드 + 3개 차트) |
| 2 | 기간 탭 변경 | 선택한 기간의 데이터 로드, 요약 카드 + 차트 갱신 |
| 3 | 차트 위 마우스 호버 | 해당 날짜의 정확한 수치 툴팁 표시 |
| 4 | 요약 카드 영역 (정보성) | 클릭 동작 없음 (조회 전용) |

### 기간별 표시 차이

| 기간 | X축 간격 | 비교 기간 |
|------|----------|-----------|
| 7일 | 일별 (7개 포인트) | vs 이전 7일 |
| 30일 | 일별 (30개 포인트) | vs 이전 30일 |
| 90일 | 주별 (13개 포인트, 주 단위 집계) | vs 이전 90일 |

### 변화율 계산

- **업로드 수 / 공유 링크**: (현재 기간 합계 - 이전 기간 합계) / 이전 기간 합계 × 100 (%)
- **활성 사용자**: 현재 기간 고유 사용자 수 - 이전 기간 고유 사용자 수 (절대값 차이, "±N명")
- 이전 기간 데이터가 없으면 변화율 대신 "—" 표시

## 반응형 동작

| 브레이크포인트 | 변경 사항 |
|----------------|-----------|
| Desktop (1024px+) | Sidebar 240px + 요약 카드 3열 + 차트 전체 너비 |
| Tablet (768~1023px) | Sidebar 접힘, 요약 카드 3열 유지, 차트 전체 너비 |
| Mobile (< 768px) | Sidebar 숨김, 요약 카드 1열(세로 스택), 차트 가로 스크롤 가능 |

## 접근성

- 요약 카드: `aria-label` (예: "총 업로드 수 156건, 이전 7일 대비 12% 증가")
- 변화율: 색상 외 방향 화살표(↑↓→) 텍스트 병행
- 바 차트: `role="img"`, `aria-label` (데이터 요약 텍스트)
- 라인 차트: `role="img"`, `aria-label` (추이 요약 텍스트)
- 기간 탭: `role="tablist"`, `role="tab"`, `aria-selected`
- 데이터 없음 상태: `role="status"`
