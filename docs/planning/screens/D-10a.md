# D-10a 디바이스 관리 (설정 > 보안)

## 화면 개요

- 목적: 등록된 2FA 디바이스 목록 조회, 추가, 제거, 주 디바이스 설정
- 진입 경로: `/settings/security` (프로필 설정 > 보안 탭)
- 사용자: 로그인된 사용자
- 레이아웃: **Layout-A** (Drive Web 기본) — 설정 서브페이지
- 플랫폼: Web + 모바일 앱 공용

## 레이아웃 구조 (메인 콘텐츠 영역)

```
┌─────────────────────────────────────────────────┐
│ 보안 설정                                        │
│                                                  │
│ 등록된 디바이스                                   │
│ ┌──────────────────────────────────────────────┐ │
│ │ ★ iPhone 15          iOS     등록: 2026-03-01│ │
│ │                      주 디바이스    [제거]    │ │
│ ├──────────────────────────────────────────────┤ │
│ │   Galaxy S24         Android  등록: 2026-03-10│ │
│ │                      [주 디바이스로] [제거]   │ │
│ └──────────────────────────────────────────────┘ │
│                                                  │
│ 백업 코드                                        │
│ 남은 코드: 6 / 8                                 │
│ [백업 코드 재발급]                               │
│                                                  │
│ 신뢰된 기기                                      │
│ ┌──────────────────────────────────────────────┐ │
│ │ Chrome · Windows     마지막 접속: 2026-03-20 │ │
│ │ 만료: 2026-04-19                    [해제]   │ │
│ └──────────────────────────────────────────────┘ │
│                                                  │
└─────────────────────────────────────────────────┘
```

## 구성 요소

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 섹션 제목 | Text (h2) | "보안 설정" | - |
| 디바이스 목록 | List | 등록된 2FA 디바이스 (이름, 플랫폼, 등록일) | 주 디바이스(★ 표시) / 일반 |
| 주 디바이스 설정 | Button (text) | 해당 디바이스를 주 디바이스로 변경 | 기본 / 호버 |
| 디바이스 제거 | Button (danger, text) | 디바이스 등록 해제 | 기본 / 확인 다이얼로그 |
| 백업 코드 현황 | Text | 남은 코드 수 표시 | 정상(4+) / 경고(1~3, 노란색) / 위험(0, 빨간색) |
| 백업 코드 재발급 | Button (secondary) | 기존 코드 무효화 + 새 8개 발급 | 기본 / 확인 다이얼로그 |
| 신뢰된 기기 목록 | List | 2FA 스킵 중인 기기 (브라우저, OS, 만료일) | - |
| 신뢰 해제 | Button (danger, text) | 해당 기기의 신뢰 상태 해제 | 기본 / 확인 다이얼로그 |

## 인터랙션

| 사용자 액션 | 시스템 반응 |
|-------------|-------------|
| 주 디바이스 변경 클릭 | 확인 다이얼로그 → 승인 시 주 디바이스 변경 |
| 디바이스 제거 클릭 | 확인 다이얼로그 ("이 디바이스에서 2FA를 받을 수 없게 됩니다") → 제거 |
| 마지막 디바이스 제거 시도 | 차단: "최소 1개의 디바이스가 필요합니다" |
| 백업 코드 재발급 클릭 | 확인 다이얼로그 ("기존 코드가 모두 무효화됩니다") → 새 코드 8개 표시 |
| 신뢰 해제 클릭 | 해당 기기에서 다음 로그인 시 2FA 필요 |
