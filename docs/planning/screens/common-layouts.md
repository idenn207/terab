# 공통 레이아웃 패턴

## Layout-A: Drive Web 기본 레이아웃

> 적용 화면: D-03 ~ D-09 (인증 완료 후 모든 Drive 화면)

```
┌──────────────────────────────────────────────────────────┐
│ Header (56px)                                            │
│ ┌──────┐  ┌─────────────────────────┐  ┌──┐ ┌─────────┐ │
│ │ Logo │  │ 🔍 검색바               │  │🔔│ │ 프로필  │ │
│ └──────┘  └─────────────────────────┘  └──┘ └─────────┘ │
├────────────┬─────────────────────────────────────────────┤
│ Sidebar    │ Main Content                                │
│ (240px)    │                                             │
│            │ ┌─ Toolbar ──────────────────────────────┐  │
│ 내 드라이브│ │ 브레드크럼  │  정렬 · 보기모드 · 업로드│  │
│ 공유 항목  │ └────────────────────────────────────────┘  │
│ 즐겨찾기   │                                             │
│ 최근 파일  │ ┌─ Content Area ─────────────────────────┐  │
│ 휴지통     │ │                                        │  │
│            │ │  파일/폴더 목록 (그리드 또는 리스트)    │  │
│            │ │                                        │  │
│ ─────────  │ │                                        │  │
│ 스토리지   │ │                                        │  │
│ ███░░ 60%  │ │                                        │  │
│ 6GB / 10GB │ └────────────────────────────────────────┘  │
├────────────┴─────────────────────────────────────────────┤
```

**구성 요소:**

| 영역 | 너비/높이 | 설명 |
|------|-----------|------|
| Header | 100% × 56px | 로고, 검색바, 알림, 프로필 메뉴 |
| Sidebar | 240px × 나머지 | 네비게이션 메뉴 + 스토리지 사용량 바 |
| Main Content | 나머지 × 나머지 | Toolbar + Content Area |
| Toolbar | 100% × 48px | 브레드크럼, 정렬, 보기 모드 전환, 업로드 버튼 |
| Content Area | 나머지 | 파일/폴더 목록 (그리드/리스트 뷰) |

## Layout-B: Admin Web 기본 레이아웃

> 적용 화면: A-02 ~ A-15 (인증 완료 후 모든 Admin 화면)

```
┌──────────────────────────────────────────────────────────┐
│ Header (56px)                                            │
│ ┌──────────────┐                        ┌─────────────┐  │
│ │ NAS Drive    │                        │ 관리자 이름  │  │
│ │ Admin        │                        │ 로그아웃     │  │
│ └──────────────┘                        └─────────────┘  │
├────────────┬─────────────────────────────────────────────┤
│ Sidebar    │ Main Content                                │
│ (240px)    │                                             │
│            │ ┌─ Page Header ──────────────────────────┐  │
│ 대시보드   │ │ 페이지 제목          [액션 버튼]       │  │
│ 사용자 관리│ └────────────────────────────────────────┘  │
│ 역할/권한  │                                             │
│ 스토리지   │ ┌─ Content Area ─────────────────────────┐  │
│ 공유 관리  │ │                                        │  │
│ 시스템     │ │  페이지별 콘텐츠                       │  │
│ 설정       │ │  (테이블, 차트, 폼 등)                 │  │
│ 감사 로그  │ │                                        │  │
│            │ │                                        │  │
│            │ └────────────────────────────────────────┘  │
├────────────┴─────────────────────────────────────────────┤
```

**구성 요소:**

| 영역 | 너비/높이 | 설명 |
|------|-----------|------|
| Header | 100% × 56px | 서비스명("NAS Drive Admin"), 관리자 프로필 |
| Sidebar | 240px × 나머지 | 8개 관리 메뉴 (IA 참조) |
| Page Header | 100% × 56px | 페이지 제목 + 주요 액션 버튼 |
| Content Area | 나머지 | 페이지별 콘텐츠 |

## Layout-C: 인증 페이지 레이아웃

┌──────────────────────────────────────────────────────────┐
│                                                          │
│                                                          │
│              ┌──────────────────────┐                    │
│              │     NAS Drive        │                    │
│              │     ──────────       │                    │
│              │                      │                    │
│              │  [ 입력 필드 ]       │                    │
│              │  [ 입력 필드 ]       │                    │
│              │                      │                    │
│              │  [ 로그인 버튼 ]     │                    │

│              └──────────────────────┘                    │

│                                                          │
