# D-03 드라이브 메인

## 화면 개요

- 목적: 루트 드라이브의 파일/폴더 목록 조회 및 파일 관리 (업로드, 다운로드, CRUD, 이동/복사, 삭제, 검색)
- 진입 경로: `/drive` (로그인 성공 후 기본 랜딩, 사이드바 "내 드라이브" 클릭)
- 사용자: 로그인된 모든 사용자 (페르소나: 민수, 지은)
- 레이아웃: **Layout-A** (Drive Web 기본)
- 플랫폼: Web + 모바일 앱 공용
- 관련 요구사항: FILE-01 ~ FILE-11

## 구성 요소

### Toolbar 영역

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 브레드크럼 | Breadcrumb | "내 드라이브" 고정 텍스트 (D-03에서는 단일 세그먼트) | - |
| 정렬 드롭다운 | Select | 이름/수정일/크기 + 오름차순/내림차순 | 기본(이름 오름차순) |
| 보기 모드 토글 | ToggleButton | 그리드/리스트 뷰 전환 (설정 localStorage 저장) | 그리드(기본) / 리스트 |
| 업로드 버튼 | Button (primary) | 클릭 시 파일 선택 다이얼로그 (다중 선택 가능) | 기본 / 호버 |
| 새 폴더 버튼 | Button (secondary) | 새 폴더 생성 모달 열기 | 기본 / 호버 |

### Content Area — 그리드 뷰

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 파일/폴더 카드 | Card | 아이콘 + 이름 (파일: + 확장자) | 기본 / 호버(그림자) / 선택(파란 테두리) |
| 체크박스 | Checkbox | 호버 시 카드 좌상단에 표시, 멀티셀렉트용 | 숨김(기본) / 표시(호버·선택 시) / 체크 |
| 더보기 버튼 | IconButton (⋮) | 호버 시 카드 우상단에 표시, 컨텍스트 메뉴 열기 | 숨김(기본) / 표시(호버) |

### Content Area — 리스트 뷰

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 테이블 헤더 | TableHeader | 전체 선택 체크박스 · 이름 · 크기 · 수정일 · 액션 | 정렬 가능 컬럼 (클릭 시 정렬 변경) |
| 파일/폴더 행 | TableRow | 체크박스 + 아이콘 + 이름 + 크기 + 수정일 + ⋮ | 기본 / 호버(배경 변경) / 선택(파란 배경) |

### Selection Toolbar (선택 시 Toolbar 대체)

> 1개 이상 항목 선택 시 기본 Toolbar를 대체하여 표시

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 선택 해제 버튼 | IconButton (✕) | 선택 해제, 기본 Toolbar로 복귀 | 기본 |
| 선택 카운트 | Text | "N개 선택됨" | - |
| 다운로드 | Button | 단일: 직접 다운로드 / 다중: ZIP 묶음 | 기본 |
| 복사 | Button | 복사 대상 폴더 선택 모달 열기 | 기본 |
| 이동 | Button | 이동 대상 폴더 선택 모달 열기 | 기본 |
| 공유 | Button | 공유 설정 (v0.1.2 구현 예정) | 비활성(v0.1.0) / 기본(v0.1.2) |
| 삭제 | Button (danger) | 삭제 확인 모달 열기 | 기본 |

### 오버레이/모달

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 드래그 오버레이 | Overlay | 파일 드래그 진입 시 Content Area 위에 표시 | 비활성(기본) / 활성(드래그 진입) |

## 모달/오버레이 상세

### 새 폴더 모달

```
┌──────────────────────────────────┐
│ 새 폴더                      ✕  │
│                                  │
│ 폴더 이름                        │
│ ┌──────────────────────────────┐ │
│ │ 새 폴더                     │ │
│ └──────────────────────────────┘ │
│                                  │
│              [취소]  [만들기]     │
└──────────────────────────────────┘
```

- 열릴 때 입력 필드에 "새 폴더" 기본값, 텍스트 전체 선택 상태
- 빈 이름 → 만들기 버튼 비활성
- Enter 키 → 만들기와 동일
- 중복 이름 → 입력 필드 아래 인라인 에러 메시지

### 이동/복사 모달

```
┌──────────────────────────────────┐
│ "memo.txt" 이동              ✕   │
│                                  │
│ 📁 내 드라이브                   │
│   📁 사진                        │
│   📁 문서                        │
│     📁 업무        ← 선택됨 (파란 배경) │
│     📁 개인                      │
│   📁 프로젝트                    │
│                                  │
│ [+ 새 폴더]      [취소]  [이동]  │
└──────────────────────────────────┘
```

- 폴더 트리 재귀적 표시, 클릭 시 하위 폴더 지연 로딩
- 현재 위치와 동일 폴더 → 선택 불가 (비활성 표시)
- 자기 자신의 하위 폴더로 이동 불가 (순환 참조 방지)
- "새 폴더" 버튼으로 모달 내에서 즉석 폴더 생성 가능
- 복사 시 제목("복사")과 버튼 텍스트("복사")만 변경
- 다중 선택 시 제목: "3개 항목 이동"

### 삭제 확인 모달

```
┌──────────────────────────────────┐
│ 삭제 확인                    ✕   │
│                                  │
│ "memo.txt"을(를) 휴지통으로      │
│ 이동하시겠습니까?                │
│                                  │
│              [취소]  [삭제]       │
└──────────────────────────────────┘
```

- 삭제 버튼: danger 스타일 (빨간색 계열)
- 다중 선택 시: "3개 항목을 휴지통으로 이동하시겠습니까?"
- 폴더 삭제 시: 하위 항목 포함 안내 ("폴더 내 모든 파일이 함께 삭제됩니다")

### 업로드 진행 패널

- 위치: 하단 우측 고정, 320px 너비
- 헤더: "업로드 중 (완료수/전체수)" + 최소화(▼) + 닫기(✕) 버튼
- 파일별 행: 아이콘 + 이름 + 진행률 바 + 퍼센트 + 취소(✕)
- 완료된 파일: ✅ + 이름 + "완료"
- 실패한 파일: ❌ + 이름 + "재시도" 버튼
- 대용량 파일: 진행률 바 아래 "청크 N/M" 표시 (FILE-11)
- 전체 완료 시: "N개 파일 업로드 완료" 토스트 → 3초 후 패널 자동 닫힘
- 페이지 이동해도 패널 유지 (업로드 진행은 글로벌 상태)
- 최소화 시: 헤더만 표시 ("업로드 중 2/5")

## 반응형 동작

| 브레이크포인트 | 변경 사항 |
|----------------|-----------|
| Desktop (1024px+) | Sidebar 240px 고정 + Main Content 가변, 그리드 뷰 4~6열 |
| Tablet (768~1023px) | Sidebar 접힘 (햄버거 메뉴 토글), 그리드 뷰 3~4열 |
| Mobile (< 768px) | Sidebar 완전 숨김 (바텀 네비게이션 또는 햄버거), 그리드 뷰 2열, 리스트 뷰 크기/수정일 컬럼 숨김, 드래그 앤 드롭 비활성 (업로드 버튼만), 컨텍스트 메뉴 대신 바텀 시트 |

## 접근성

- 파일/폴더 목록: `role="grid"` (그리드 뷰) / `role="table"` (리스트 뷰)
- 키보드 탐색: 화살표 키로 항목 이동, Enter로 열기, Space로 선택 토글
- 스크린리더: 각 항목에 aria-label (이름, 크기, 수정일 포함)
- 드래그 앤 드롭: 키보드 대안으로 업로드 버튼 제공
- 컨텍스트 메뉴: `aria-haspopup="menu"`, 포커스 트랩, Esc로 닫기
- 업로드 진행: `role="progressbar"`, `aria-valuenow` 동적 업데이트
- 색상 대비: WCAG AA 기준 준수
