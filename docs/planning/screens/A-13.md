# A-13 서비스 기본값

## 화면 개요

- 목적: 서비스 전역 기본값 설정 (기본 쿼터, 공유 링크 만료일, 휴지통 보관 기간)
- 진입 경로: `/settings/defaults` (사이드바 "설정" → 서브메뉴 "기본값")
- 사용자: `system:config` 권한 보유 관리자 (페르소나: 민수)
- 레이아웃: **Layout-B** (Admin Web 기본)
- 플랫폼: PC Web 전용
- 관련 요구사항: ADMIN-14

## 인터랙션 상세

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 페이지 진입 | 현재 설정값 로드 (기본 쿼터, 만료일, 보관 기간) |
| 2 | 필드 값 변경 | 변경 감지 → 취소/저장 버튼 활성화 |
| 3 | 단위 셀렉터 변경 | MB ↔ GB 전환, 수치는 그대로 유지 (단위만 변경) |
| 4 | 저장 클릭 | 유효성 검증 → 성공 시 API 호출 → 성공 Toast "서비스 기본값이 저장되었습니다" |
| 5 | 취소 클릭 | 모든 필드를 서버에서 로드한 초기값으로 복원 |
| 6 | 변경 사항 있는 상태에서 페이지 이탈 | "저장하지 않은 변경 사항이 있습니다. 이동하시겠습니까?" 확인 다이얼로그 |

### 유효성 검증

| 필드 | 규칙 | 에러 메시지 |
|------|------|-------------|
| 기본 쿼터 | 1 이상의 정수 | "1 이상의 숫자를 입력해주세요" |
| 공유 링크 만료일 | 0 이상의 정수 | "0 이상의 숫자를 입력해주세요" |
| 휴지통 보관 기간 | 0 이상의 정수 | "0 이상의 숫자를 입력해주세요" |

### 설정 적용 범위

> 기본값 변경은 **변경 이후 새로 생성되는 항목**에만 적용됩니다.
> - 기본 쿼터: 변경 후 가입하는 신규 사용자에게 적용. 기존 사용자의 쿼터는 A-04에서 개별 변경.
> - 공유 링크 만료일: 변경 후 생성하는 공유 링크에 적용. 기존 링크는 영향 없음.
> - 휴지통 보관 기간: 변경 후 삭제되는 파일에 적용. 기존 휴지통 파일은 기존 기간 유지.

## 접근성

- 모든 입력 필드: `<label>` 연결, `aria-describedby` (설명 텍스트 연결)
- 단위 셀렉터: `aria-label="용량 단위"`, 키보드 탐색 가능
- 섹션 구분: `<fieldset>` + `<legend>`로 그룹핑
- 에러 메시지: `role="alert"`, `aria-live="assertive"`
- 저장 성공 Toast: `aria-live="polite"`
