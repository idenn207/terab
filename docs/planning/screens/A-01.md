# A-01 관리자 로그인

## 화면 개요

- 목적: 관리자 인증 (ID/PW + 2FA)
- 진입 경로: `admin.drive.skypark207.com/login` (미인증 상태에서 모든 Admin URL 접근 시 리다이렉트)
- 사용자: ADMIN 역할 보유 사용자 (페르소나: 민수)
- 레이아웃: **Layout-C** (인증 페이지)
- 플랫폼: PC Web 전용
- 관련 요구사항: AUTH-01, AUTH-02, AUTH-03

## D-01 대비 차이점

> 기본 구조와 인터랙션은 [D-01 로그인](D-01.md)과 동일합니다. 아래는 차이점만 기술합니다.

### 브랜딩

| 요소 | D-01 (Drive) | A-01 (Admin) |
|------|-------------|-------------|
| 서비스 로고 | "🗂 NAS Drive" | "🛡 NAS Drive Admin" |
| 배경색 | 기본 | 기본과 동일 (또는 Admin 식별 컬러 적용 가능) |

### 제거된 요소

| 요소 | 사유 |
|------|------|
| 이 기기를 신뢰 (체크박스) | Admin은 보안 강화를 위해 항상 2FA 수행 |
| 회원가입 링크 | 관리자 계정은 기존 관리자가 초대(A-05)로 생성 |
| 모바일 앱 생체인증 | Admin은 PC Web 전용 |

### 변경된 인터랙션

| # | 사용자 액션 | D-01 | A-01 |
