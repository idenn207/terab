# A-05 사용자 초대

## 화면 개요

- 목적: 초대 링크를 생성하여 새 사용자를 서비스에 초대하고, 기존 초대 현황 관리
- 진입 경로: `/users/invite` (A-03 Page Header "사용자 초대" 버튼, 또는 A-02 대시보드 빠른 액션)
- 사용자: `user:invite` 권한 보유 관리자 (페르소나: 민수)
- 레이아웃: **Layout-B** (Admin Web 기본)
- 플랫폼: PC Web 전용
- 관련 요구사항: ADMIN-02
- 관련 유저플로우: Flow 6 — 사용자 초대 링크 생성

## 구성 요소

### Page Header

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 뒤로가기 | Link | "← 사용자 목록" → A-03으로 이동 | 기본 |
| 페이지 제목 | Text (h1) | "✉ 사용자 초대" | - |

### 초대 생성 섹션

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 섹션 제목 | Text (h2) | "새 초대 링크 생성" | - |
| 역할 선택 | Select | 초대할 사용자의 기본 역할 (USER 기본값, ADMIN 선택 가능) | 기본 |
| 만료일 | DatePicker | 초대 링크 만료일 (기본값: 7일 후) | 기본 / 포커스 |
| 초대 링크 생성 버튼 | Button (primary) | API 호출하여 초대 링크 생성 | 기본 / 호버 / 로딩 |

### 생성 결과

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 성공 메시지 | Alert (success) | "✅ 초대 링크가 생성되었습니다" | 초대 생성 후 표시 |
| 초대 URL | Input (readonly, monospace) | 생성된 초대 링크 전체 URL | 읽기 전용 |
| 링크 복사 버튼 | Button (secondary) | URL 클립보드 복사 | 기본 / 클릭 후 "복사됨!" (2초) |
| 안내 텍스트 | Text (secondary) | "이 링크를 초대할 사용자에게 전달해 주세요." + 만료일 | - |

### 초대 내역

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 섹션 제목 | Text (h2) | "기존 초대 목록" | - |
| 테이블 헤더 | TableHeader | 생성일 · 역할 · 만료일 · 상태 · 액션 | - |
| 초대 행 | TableRow | 초대 정보 1행 | 기본 |
| 상태 배지 | Badge | 🟡 대기 / 🟢 사용됨 / 🔴 만료 | 색상 구분 |
| 삭제 버튼 | IconButton (🗑) | 대기/만료 상태의 초대 삭제 | 기본 / 사용됨 상태에서는 미표시 |

## 인터랙션 상세

### 초대 생성

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 역할 선택 + 만료일 설정 → 초대 링크 생성 클릭 | API 호출 → 성공: 생성 결과 영역 표시, 초대 내역 테이블에 추가 |
| 2 | 링크 복사 클릭 | URL 클립보드 복사, 버튼 텍스트 "복사됨!" (2초 후 원복) |
| 3 | OWNER 역할로 초대 시도 | OWNER는 선택 불가 (드롭다운에서 제외, 시스템에 1명만 존재) |

### 초대 관리

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 4 | 삭제 버튼 클릭 (대기 상태) | 삭제 확인: "이 초대 링크를 삭제하시겠습니까?" → 확인 시 삭제, 토스트: "초대가 삭제되었습니다" |
| 5 | 삭제 버튼 클릭 (만료 상태) | 즉시 삭제 (확인 없이), 토스트: "만료된 초대가 삭제되었습니다" |

## 상태 목록

| # | 상태 | 설명 |
|---|------|------|
| 1 | 기본 | 초대 생성 폼 + 기존 초대 목록 표시 |
| 2 | 생성 성공 | 초대 폼 + 생성 결과 영역 + 초대 내역 |
| 3 | 빈 상태 | 초대 내역이 없을 때: "생성된 초대가 없습니다" |
| 4 | 로딩 | 초대 내역 스켈레톤 |
| 5 | 에러 | API 오류 시 토스트 에러 메시지 |

## 접근성

- 폼 필드: `<label>` 연결, `aria-required`
- 생성 결과: `role="alert"`, 스크린리더 즉시 읽기
- 복사 버튼: `aria-live="polite"` (복사 완료 상태 변경 알림)
- 초대 테이블: `role="table"`, 각 행에 aria-label
- 상태 배지: 색상 외에 텍스트("대기"/"사용됨"/"만료")로 상태 전달
