# D-02 회원가입 (모바일 앱 우선)

## 화면 개요

- 목적: 초대 코드 기반 신규 사용자 등록 + 디바이스 자동 등록
- 진입 경로: 초대 링크 딥링크 → 앱 실행 → 가입 화면 (앱 미설치 시 앱스토어로 이동)
- 사용자: 초대를 받은 신규 사용자 (페르소나: 지은)
- 레이아웃: **Layout-C** (인증 페이지)
- 플랫폼: **모바일 앱 우선** (PC Web에서도 접근 가능하나 앱 설치 유도)

## 온보딩 시나리오

```
초대 링크 클릭 (drive.skypark207.com/invite/abc123)
         │
    ┌────┴─────┐
    │          │
  앱 설치됨  앱 미설치
    │          │
  딥링크로   앱스토어 이동
  앱에서     (스마트 배너)
  가입 폼      │
  표시       설치 후
             링크 재클릭
```

## 구성 요소

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 서비스 로고 | 이미지/텍스트 | "NAS Drive" + "회원가입" 부제 | - |
| 아이디 입력 | Input (text) | 로그인용 고유 ID (4~20자, 영문소문자+숫자+밑줄) | 기본 / 포커스 / 에러 / 중복 에러 |
| 닉네임 입력 | Input (text) | 서비스 내 표시명 (2~20자) | 기본 / 포커스 / 에러 |
| 이메일 입력 | Input (email) | **선택 항목** — 알림 수신용 (라벨에 "(선택)" 표시) | 기본 / 포커스 / 에러 |
| 비밀번호 입력 | Input (password) | 비밀번호 (8자 이상, 영문+숫자+특수문자) | 기본 / 포커스 / 에러 / 강도 표시 |
| 비밀번호 확인 | Input (password) | 비밀번호 재입력 | 기본 / 포커스 / 불일치 에러 |
| 비밀번호 강도 | ProgressBar | 약함/보통/강함 3단계 색상 바 | 빨강/노랑/초록 |
| 가입하기 버튼 | Button (primary) | 회원가입 API 호출 | 기본 / 호버 / 로딩 / 비활성 |
| 로그인 링크 | TextLink | `/login`으로 이동 | 기본 / 호버 |
| 초대 코드 만료 안내 | FullScreenAlert | 유효하지 않은 초대 코드일 때 전체 화면 대체 표시 | "초대 링크가 만료되었거나 유효하지 않습니다" |

## 인터랙션

| 사용자 액션 | 시스템 반응 |
|-------------|-------------|
| 딥링크로 앱 실행 | 초대 코드 추출 → 유효성 검증 → 실패 시 만료 안내 |
| 모든 필수 필드 입력 후 가입 클릭 | 유효성 검증 → API 호출 → 계정 생성 (USER 역할 기본 부여) |
| 가입 성공 | **자동 로그인 + 디바이스 자동 등록** (Push 토큰 서버 전송) → 백업 코드 8개 표시 화면으로 이동 |
| 비밀번호 입력 중 | 실시간 강도 표시 (약함/보통/강함) |
| 비밀번호 확인 불일치 | 즉시 에러 메시지 표시 |
| 아이디 중복 | API 응답 후 에러 표시 ("이미 사용 중인 아이디입니다") |
| PC Web에서 초대 링크 접속 | "모바일 앱에서 가입하세요" 안내 + 앱스토어 QR코드/링크 표시 |

## 유효성 검증 규칙

| 필드 | 규칙 | 에러 메시지 |
|------|------|-------------|
| 아이디 | 4~20자, 영문소문자+숫자+밑줄만 허용 | "아이디는 4~20자, 영문 소문자·숫자·밑줄만 사용 가능합니다" |
| 닉네임 | 2~20자 | "닉네임은 2~20자여야 합니다" |
| 이메일 | 입력 시 이메일 형식 검증 (미입력 허용) | "올바른 이메일 형식이 아닙니다" |
| 비밀번호 | 8자 이상, 영문+숫자+특수문자 | "비밀번호는 8자 이상, 영문·숫자·특수문자를 포함해야 합니다" |
| 비밀번호 확인 | 비밀번호와 일치 | "비밀번호가 일치하지 않습니다" |

## 반응형 동작

- 모바일 앱: 전체 화면, 스크롤 가능
- PC Web: 카드 400px 중앙 정렬 + "앱에서 가입하세요" 안내 표시
