# D-02a 백업 코드 발급 (가입 직후)

## 화면 개요

- 목적: 회원가입 직후 백업 코드 8개를 표시하고 사용자에게 안전하게 보관하도록 안내
- 진입 경로: D-02 가입 성공 후 자동 전환
- 사용자: 방금 가입한 사용자
- 플랫폼: **모바일 앱** (가입이 앱에서 이루어지므로)

```
┌────────────────────────┐
│                        │
│    2FA 설정 완료!       │
│                        │

│                        │
│    아래 백업 코드를     │
│    안전한 곳에          │

│                        │
│    ┌──────────────────┐│
│    │ ABCD-1234        ││
```

## 구성 요소

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 완료 안내 | Text (h2) | "2FA 설정 완료!" | - |
| 디바이스 등록 안내 | Text | 현재 디바이스가 2FA 디바이스로 등록됨을 안내 | - |
| 백업 코드 목록 | Code Block | 8개의 1회용 백업 코드 (형식: XXXX-XXXX) | - |
| 복사하기 버튼 | Button (secondary) | 전체 코드를 클립보드에 복사 | 기본 / 복사 완료("복사됨!") |
| 경고 문구 | Alert (warning) | "이 코드는 다시 확인할 수 없습니다" | - |
| 시작하기 버튼 | Button (primary) | `/drive`로 이동 | 기본 / 호버 |

## 인터랙션

| 사용자 액션 | 시스템 반응 |
|-------------|-------------|
| 복사하기 클릭 | 클립보드에 8개 코드 복사, 버튼 텍스트 "복사됨!" 전환 |
| 시작하기 클릭 | `/drive` (드라이브 메인)으로 이동 |
| 뒤로가기 | 차단 (이 화면을 건너뛸 수 없음) |
