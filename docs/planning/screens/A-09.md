# A-09 공유 링크 관리

## 화면 개요

- 목적: 서비스 전체의 활성 공유 링크를 조회하고, 문제 있는 링크를 강제 만료 처리
- 진입 경로: `/shares` (사이드바 "공유 관리" 클릭)
- 사용자: `share:manage` 권한 보유 관리자 (페르소나: 민수)
- 레이아웃: **Layout-B** (Admin Web 기본)
- 플랫폼: PC Web 전용
- 관련 요구사항: ADMIN-09, ADMIN-10

## 인터랙션 상세

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 페이지 진입 | 전체 공유 링크 목록 로드 (기본: 생성일 내림차순, 상태 전체) |

| Mobile (< 768px) | Sidebar 숨김, 필터 세로 스택, "생성일"/"접근 수" 컬럼 숨김, 테이블 가로 스크롤 |

## 접근성

- 테이블: `role="table"`, 정렬 상태 `aria-sort`
- 상태 배지: 색상 외 텍스트 라벨 병행 ("활성", "만료", "강제 만료")
- 강제 만료 버튼: `aria-label="[파일명] 공유 링크 강제 만료"`
- 모달: 포커스 트랩, Esc로 닫기, `role="alertdialog"`, `aria-describedby` (경고 메시지)
- 검색 입력: `<label>` 연결, `aria-describedby`
- 필터 변경 시 결과 업데이트: `aria-live="polite"` (결과 건수 안내)
