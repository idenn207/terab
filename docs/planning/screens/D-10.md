# D-10 프로필 설정

## 화면 개요

- 목적: 사용자 프로필(닉네임) 변경 및 비밀번호 변경
- 진입 경로: `/settings/profile` (헤더 프로필 메뉴 > "설정", 또는 사이드바 하단)
- 사용자: 로그인된 모든 사용자 (페르소나: 민수, 지은)
- 레이아웃: **Layout-A** (Drive Web 기본) — 설정 서브페이지
- 플랫폼: Web + 모바일 앱 공용
- 관련 요구사항: AUTH-04

## 구성 요소

### 설정 탭 바

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 프로필 탭 | Tab | `/settings/profile` (현재 화면) | 활성 |
| 보안 탭 | Tab | `/settings/security` (D-10a) | 비활성 |

### 프로필 정보 섹션

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 섹션 제목 | Text (h2) | "프로필 정보" | - |
| 프로필 이미지 | Avatar | 현재 프로필 이미지 (기본: 이니셜 아바타) | - |
| 이미지 변경 버튼 | Button (text) | 이미지 선택 다이얼로그 (JPG/PNG, 최대 2MB) | 기본 / 호버 |
| 닉네임 입력 | Input | 현재 닉네임 표시, 수정 가능 | 기본 / 포커스 / 에러 |
| 이메일 | Input (disabled) | 가입 시 등록한 이메일, 변경 불가 | 비활성(회색 배경) |
| 저장 버튼 | Button (primary) | 닉네임/이미지 변경 저장 | 기본 / 비활성(변경사항 없을 때) / 로딩 |

### 비밀번호 변경 섹션

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 섹션 제목 | Text (h2) | "비밀번호 변경" | - |
| 현재 비밀번호 | Input (password) | 현재 비밀번호 입력 | 기본 / 포커스 / 에러 |
| 새 비밀번호 | Input (password) | 새 비밀번호 입력 + 강도 표시 | 기본 / 포커스 / 에러 |
| 비밀번호 확인 | Input (password) | 새 비밀번호 재입력 | 기본 / 포커스 / 에러(불일치) |
| 비밀번호 변경 버튼 | Button (primary) | 비밀번호 변경 API 호출 | 기본 / 비활성(필드 미입력) / 로딩 |

## 인터랙션 상세

### 프로필 정보

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 닉네임 수정 | 저장 버튼 활성화 |
| 2 | 저장 클릭 | API 호출 → 성공: 토스트 "프로필이 저장되었습니다", 헤더 프로필 메뉴 이름 갱신 |
| 3 | 닉네임 빈 값 | 인라인 에러: "닉네임을 입력하세요" |
| 4 | 닉네임 길이 초과 (20자) | 인라인 에러: "닉네임은 20자 이내로 입력하세요" |
| 5 | 프로필 이미지 변경 클릭 | 파일 선택 다이얼로그 (JPG/PNG, 2MB 이하) |
| 6 | 이미지 선택 | 미리보기 표시, 저장 버튼 활성화 |
| 7 | 허용되지 않은 파일 형식 또는 크기 초과 | 토스트 에러: "JPG/PNG 파일만 업로드 가능합니다 (최대 2MB)" |

### 비밀번호 변경

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 8 | 3개 필드 모두 입력 | 비밀번호 변경 버튼 활성화 |
| 9 | 비밀번호 변경 클릭 | 현재 비밀번호 검증 → 성공: 토스트 "비밀번호가 변경되었습니다", 입력 필드 초기화 |
| 10 | 현재 비밀번호 불일치 | 인라인 에러: "현재 비밀번호가 일치하지 않습니다" |
| 11 | 새 비밀번호 규칙 미충족 | 인라인 에러: "비밀번호는 8자 이상, 영문+숫자+특수문자 포함" |
| 12 | 비밀번호 확인 불일치 | 인라인 에러: "비밀번호가 일치하지 않습니다" |
| 13 | 새 비밀번호 입력 시 | 비밀번호 강도 표시: 약함(빨강) / 보통(노랑) / 강함(초록) |

## 상태 목록

| # | 상태 | 설명 |
|---|------|------|
| 1 | 기본 | 현재 프로필 정보 로드, 비밀번호 필드 비어있음 |
| 2 | 로딩 | 프로필 정보 로드 중 (스켈레톤 UI) |
| 3 | 저장 중 | 저장/변경 버튼 로딩 스피너 |
| 4 | 에러 | API 호출 실패 시 토스트 에러 메시지 |

## 반응형 동작

| 브레이크포인트 | 변경 사항 |
|----------------|-----------|
| Desktop (1024px+) | 폼 max-width 600px, 좌측 정렬 |
| Tablet (768~1023px) | Sidebar 접힘, 폼 전체 너비 |
| Mobile (< 768px) | Sidebar 숨김, 폼 전체 너비, 섹션 간 간격 축소 |

## 접근성

- 폼 필드: `<label>` 연결, `aria-required`, `aria-describedby` (에러 메시지)
- 비밀번호 강도: `role="status"`, `aria-live="polite"`
- 탭 바: `role="tablist"` + `role="tab"`, `aria-selected`
- 에러 메시지: `role="alert"`, 스크린리더 즉시 읽기
