# D-05 내가 공유한 항목

## 화면 개요

- 목적: 내가 생성한 공유 링크 목록을 관리 (조회, 설정 변경, 삭제)
- 진입 경로: `/shared/by-me` (사이드바 "공유 항목" > "내가 공유한" 클릭)
- 사용자: 로그인된 모든 사용자 (페르소나: 민수, 지은)
- 레이아웃: **Layout-A** (Drive Web 기본)
- 플랫폼: Web + 모바일 앱 공용
- 관련 요구사항: SHARE-01, SHARE-02, SHARE-04
- v0.1 범위: **링크 공유만** — 사용자 권한 공유(SHARE-03)는 v0.1.2에서 구현

## 인터랙션 상세

### 링크 관리

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 링크 복사 클릭 | URL 클립보드 복사, 버튼 텍스트 "복사됨!" (2초 후 원복) |
| 2 | 설정 클릭 | 공유 설정 모달 열기 (기존 설정값 로드) |
| 3 | 삭제 클릭 | 삭제 확인 모달: "공유 링크를 삭제하면 이 링크로 더 이상 접근할 수 없습니다" |
| 4 | 삭제 확인 | 링크 삭제, 목록에서 제거, 토스트: "공유 링크가 삭제되었습니다" |
| 5 | 재활성화 클릭 (만료된 링크) | 공유 설정 모달 열기 (만료일 필드에 포커스, 새 만료일 입력 유도) |
| 6 | 파일/폴더 이름 클릭 | 원본 파일/폴더 위치로 이동 (D-03/D-04) |
| 7 | 사용자 공유 탭 클릭 | "사용자 공유 기능은 준비 중입니다" 안내 표시 |

### 새 공유 생성

- 이 화면에서 직접 새 공유 링크를 생성하지 않음
- D-03/D-04의 컨텍스트 메뉴 "🔗 공유" 또는 Selection Toolbar를 통해 생성 (v0.1.2)
- 생성 후 이 화면에 자동으로 반영

## 반응형 동작

| 브레이크포인트 | 변경 사항 |
|----------------|-----------|
| Desktop (1024px+) | 카드 전체 너비, 버튼 인라인 표시 |
| Tablet (768~1023px) | Sidebar 접힘, 카드 전체 너비 유지 |
| Mobile (< 768px) | 카드 내 URL 말줄임, 버튼 아이콘만 표시, 설정은 카드 클릭으로 접근 |

## 접근성

- 공유 링크 카드: `role="article"`, 각 카드에 aria-label (파일명 + 상태)
- 탭 바: `role="tablist"` + `role="tab"`, `aria-selected`
- 복사 버튼: `aria-live="polite"` (복사 완료 상태 변경 알림)
- 만료됨 배지: `role="status"`, 스크린리더에 "만료된 공유 링크" 안내
