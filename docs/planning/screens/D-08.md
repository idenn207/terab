# D-08 최근 파일

## 화면 개요

- 목적: 최근 열람하거나 업로드한 파일을 시간순으로 보여주는 화면
- 진입 경로: `/recent` (사이드바 "최근 파일" 클릭)
- 사용자: 로그인된 모든 사용자 (페르소나: 민수, 지은)
- 레이아웃: **Layout-A** (Drive Web 기본)
- 플랫폼: Web + 모바일 앱 공용
- 관련 요구사항: FILE-09
- 기본 동작: **D-03과 동일** — 아래 차이점만 기술

## D-03 대비 차이점

### 1. Toolbar 변경

```
┌─ Toolbar ──────────────────────────────────────────────────┐
│ 🕐 최근 파일                     정렬▼  ☷ ≡               │
└────────────────────────────────────────────────────────────┘
```

- 브레드크럼 대신 "🕐 최근 파일" 페이지 제목 표시
- **[+ 업로드] [📁 새 폴더] 버튼 없음**
- 정렬 기본값: **활동 시각 내림차순** (최신 먼저)
- 정렬 옵션: 활동 시각 / 이름 / 크기

### 2. 표시 항목

- **파일만 표시** (폴더 제외) — 최근 "열람/업로드한 파일" 목적
- 동일 파일이 여러 번 열람된 경우 **가장 최근 활동 시각만** 표시 (중복 제거)

### 3. 리스트 뷰 — 활동 시각 · 원래 위치 열 추가

```
┌─ Content Area ─────────────────────────────────────────────┐
│ ☐  이름              원래 위치           활동 시각      ⋮  │
│ ──────────────────────────────────────────────────────────│
│ ☐ 📄 report.pdf     내 드라이브 > 문서   오늘 14:30     ⋮ │
│ ☐ 🖼 photo.jpg      내 드라이브 > 사진   오늘 10:15     ⋮ │
│ ☐ 📄 memo.txt       내 드라이브          어제 18:42     ⋮ │
│ ☐ 📄 data.xlsx      내 드라이브 > 업무   3월 18일       ⋮ │
└───────────────────────────────────────────────────────────┘
```

- **활동 시각**: "수정일" 대신 마지막 열람/업로드 시각 표시
  - 오늘: "오늘 HH:mm"
  - 어제: "어제 HH:mm"
  - 올해: "M월 D일"
  - 이전: "YYYY-MM-DD"
- **원래 위치** 열: 클릭 시 해당 폴더로 이동
- 그리드 뷰에서는 카드 하단에 활동 시각 + 경로를 작은 텍스트로 표시

### 4. 날짜별 그룹 헤딩

```
── 오늘 ────────────────────────────────────
📄 report.pdf    📄 memo.txt    🖼 photo.jpg

── 어제 ────────────────────────────────────
📄 data.xlsx     📄 notes.md

── 이번 주 ─────────────────────────────────
📄 slides.pptx   🖼 banner.png
```

- 그리드 뷰에서는 **날짜별 그룹 헤딩** 표시 (오늘 / 어제 / 이번 주 / 이번 달 / 이전)
- 리스트 뷰에서는 그룹 헤딩 없이 시간순 정렬

### 5. 컨텍스트 메뉴 변경

- D-03 컨텍스트 메뉴와 동일하되:
  - "📂 원래 위치로 이동" 항목 추가
  - "📁 새 폴더" 관련 항목 없음

### 6. 빈 상태

```
┌─ Content Area ─────────────────────────────┐
│                                             │
│              🕐                             │
│       최근 활동한 파일이 없습니다             │

```
