# D-07 즐겨찾기

## 화면 개요

- 목적: 즐겨찾기에 등록된 파일/폴더 목록을 모아 보여주는 바로가기 화면
- 진입 경로: `/favorites` (사이드바 "즐겨찾기" 클릭)
- 사용자: 로그인된 모든 사용자 (페르소나: 민수, 지은)
- 레이아웃: **Layout-A** (Drive Web 기본)
- 플랫폼: Web + 모바일 앱 공용
- 관련 요구사항: FILE-08
- 기본 동작: **D-03과 동일** — 아래 차이점만 기술

## D-03 대비 차이점

### 1. Toolbar 변경

```
┌─ Toolbar ──────────────────────────────────────────────────┐
│ ⭐ 즐겨찾기                      정렬▼  ☷ ≡               │
└────────────────────────────────────────────────────────────┘
```

- 브레드크럼 대신 "⭐ 즐겨찾기" 페이지 제목 표시
- **[+ 업로드] [📁 새 폴더] 버튼 없음** — 즐겨찾기 화면에서는 새 항목 생성 불가
- 정렬/보기 모드 토글은 유지

### 2. 리스트 뷰 — 원래 위치 열 추가

```
┌─ Content Area ─────────────────────────────────────────────┐
│ ☐  이름▲                 원래 위치           수정일     ⋮  │
│ ──────────────────────────────────────────────────────────│
│ ☐ 📁 프로젝트           내 드라이브          2026-03-20  ⋮ │
│ ☐ 📄 report.pdf         내 드라이브 > 문서   2026-03-18  ⋮ │
│ ☐ 🖼 photo.jpg          내 드라이브 > 사진   2026-03-17  ⋮ │
└───────────────────────────────────────────────────────────┘
```

- **원래 위치** 열 추가: 해당 항목이 실제로 존재하는 폴더 경로 표시
- 원래 위치 클릭 → 해당 폴더로 이동 (D-04)
- 그리드 뷰에서는 카드 하단에 경로를 작은 텍스트로 표시

### 3. 컨텍스트 메뉴 변경

```
┌────────────────────────┐
│ 📂 열기                │
│ 📂 원래 위치로 이동     │
│ ──────────────────────│
│ ⬇ 다운로드             │
│ ✏ 이름 변경            │
│ 📋 복사                │
│ 📦 이동                │
│ ──────────────────────│
│ ⭐ 즐겨찾기 해제        │
│ 🔗 공유 (v0.1.2)       │
│ ──────────────────────│
│ 🗑 삭제                │
└────────────────────────┘
```

- "⭐ 즐겨찾기" → "⭐ 즐겨찾기 해제" 로 변경
- "📂 원래 위치로 이동" 항목 추가

### 4. Selection Toolbar 변경

```
┌─ Selection Toolbar ────────────────────────────────────────┐
│ ✕  3개 선택됨    [⬇ 다운로드] [⭐ 즐겨찾기 해제] [🗑 삭제] │
└────────────────────────────────────────────────────────────┘
```

- 이동/복사 버튼 유지, **즐겨찾기 해제** 버튼 추가

### 5. 인터랙션 추가

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 즐겨찾기 해제 (컨텍스트 메뉴 / Selection Toolbar) | 해당 항목 목록에서 즉시 제거, 토스트: "즐겨찾기에서 제거됨" + [실행취소] |
| 2 | 실행취소 클릭 (토스트) | 즐겨찾기 복원, 목록에 항목 다시 표시 |
| 3 | "원래 위치로 이동" 클릭 | 해당 파일이 위치한 폴더 (`/drive/:folderId`)로 이동, 항목 하이라이트 |

### 7. 그 외

- 검색: 헤더 검색바 사용 가능 (범위 토글: "즐겨찾기" / "전체 드라이브")
- 정렬, 멀티셀렉트, 키보드 단축키, 반응형, 접근성 → **D-03 참조**
- 드래그 앤 드롭 업로드: **비활성** (즐겨찾기 화면에서 업로드 불가)
