# A-03 사용자 목록

## 화면 개요

- 목적: 등록된 모든 사용자 조회, 필터링, 검색 및 사용자 상세로 진입
- 진입 경로: `/users` (사이드바 "사용자 관리" 클릭)
- 사용자: `user:read` 권한 보유 관리자 (페르소나: 민수)
- 레이아웃: **Layout-B** (Admin Web 기본)
- 플랫폼: PC Web 전용
- 관련 요구사항: ADMIN-01
- 관련 유저플로우: Flow 6 "관리자 — 사용자 및 권한 관리" — 사용자 목록 조회

## 레이아웃 구조

┌─ 필터/검색 바 ────────────────────────────────────────────┐
│ 상태: [전체▼]  역할: [전체▼]     🔍 이름 또는 이메일 검색  │
└────────────────────────────────────────────────────────────┘
┌─ 사용자 테이블 ───────────────────────────────────────────┐
│                                                            │

## 구성 요소

### Page Header

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 페이지 제목 | Text (h1) | "👥 사용자 관리" | - |
| 사용자 초대 버튼 | Button (primary) | A-05(사용자 초대)로 이동 | 기본 / 호버 |

### 필터/검색 바

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 상태 필터 | Select | 전체(기본) / 활성 / 비활성 | - |
| 역할 필터 | Select | 전체(기본) / OWNER / ADMIN / USER / 커스텀 역할 목록 | - |
| 검색 입력 | Input (search) | 이름 또는 이메일로 검색 (디바운스 300ms) | 기본 / 포커스 |

### 사용자 테이블

| 요소 | 타입 | 설명 | 상태 |
|------|------|------|------|
| 테이블 헤더 | TableHeader | 이름 · 이메일 · 상태 · 역할 · 사용량 · 가입일 · 최근 접속 | 정렬 가능 (이름, 가입일, 최근 접속, 사용량) |
| 사용자 행 | TableRow | 사용자 정보 1행 표시 | 기본 / 호버(배경 하이라이트) / 클릭 가능 |
| 이름 | Text | 사용자 닉네임 | - |
| 이메일 | Text (monospace) | 사용자 이메일 | - |
| 상태 배지 | Badge | 🟢 활성 / 🔴 비활성 | 색상으로 구분 |
| 역할 배지 | Badge (목록) | 할당된 역할들 (여러 개일 경우 배지 나열) | OWNER(보라) / ADMIN(파랑) / USER(회색) / 커스텀(초록) |
| 사용량 | Text | 현재 스토리지 사용량 | - |
| 가입일 | Text | 가입 날짜 (YYYY-MM-DD) | - |
| 최근 접속 | Text | 마지막 접속 시각 (오늘이면 시각, 아니면 날짜) | - |
| 총 사용자 수 | Text | 하단 우측 "총 N명" | - |
| 페이지네이션 | Pagination | 20건/페이지, 이전/다음 버튼, 현재 페이지 표시 | - |

## 인터랙션 상세

| # | 사용자 액션 | 시스템 반응 |
|---|-------------|-------------|
| 1 | 페이지 진입 | 전체 사용자 목록 로드 (가입일 내림차순) |
| 2 | 사용자 행 클릭 | A-04(사용자 상세 `/users/:id`)로 이동 |
| 3 | 상태 필터 변경 | 선택한 상태로 테이블 필터링, 페이지 1로 초기화 |
| 4 | 역할 필터 변경 | 선택한 역할로 테이블 필터링, 페이지 1로 초기화 |
| 5 | 검색 입력 | 이름/이메일 부분 일치로 실시간 필터링 (디바운스 300ms) |
| 6 | 테이블 헤더 클릭 (정렬) | 해당 컬럼 기준 오름차순/내림차순 토글 |
| 7 | 사용자 초대 클릭 | A-05(사용자 초대)로 이동 |
| 8 | 페이지네이션 클릭 | 해당 페이지 데이터 로드 |

## 반응형 동작

| 브레이크포인트 | 변경 사항 |
|----------------|-----------|
| Desktop (1024px+) | Sidebar 240px + 테이블 전체 컬럼 표시 |
| Tablet (768~1023px) | Sidebar 접힘, "사용량" · "가입일" 컬럼 숨김 |
| Mobile (< 768px) | Sidebar 숨김, "역할" · "최근 접속" 추가 숨김, 행 클릭 → 상세 이동 |

## 접근성

- 테이블: `role="table"`, 정렬 상태 `aria-sort`
- 상태 배지: 색상 외에 텍스트("활성"/"비활성")로 상태 전달
- 필터 드롭다운: `<label>` 연결, `aria-describedby`
- 검색 입력: `aria-label="사용자 검색"`, 검색 결과 수 `aria-live="polite"`
- 행 클릭: `role="link"` 또는 `tabindex="0"` + Enter 키 지원
